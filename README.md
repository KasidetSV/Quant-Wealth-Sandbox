## 🎯 Purpose

This repository is a lightweight, quantitative sandbox designed to model **Regime-Based Asset Allocation** and **Value-at-Risk (VaR) optimizations**. 

It demonstrates the ability to programmatically bridge advanced institutional investment theory with Python-based algorithmic execution.

## ⚙️ Core Mechanics

The included script (`eraa_simulator.py`) mocks up a simplified asset allocation engine that:
1. **Tracks Economic Regimes:** Adjusts baseline portfolio weights based on four distinct macro environments (Growth, Recession, Stagflation, Recovery).
2. **Calculates Value-at-Risk (VaR):** Computes a simplified 99% probability risk metric based on asset volatility.
3. **Optimizes to a Risk Index (SRI):** Automatically rebalances high-risk assets (like Emerging Market Equities) into protective assets (like Government Bonds) if the regime's calculated risk exceeds the client's target Risk Index.

## 🚀 How to Run

1. Ensure you have Python installed with `numpy` and `pandas`.
2. Run the simulator:
   ```bash
   python eraa_simulator.py
   ```
